## 🚀 Introduction

In today's data-driven world, the ability to interpret complex datasets is crucial. This repository serves as a portfolio of my work in **Data Analytics**, specifically focusing on **Microsoft Excel** and **Power BI**. The goal of these projects is to provide clear, visual representations of business metrics, allowing stakeholders to identify trends, outliers, and opportunities at a glance.

## 📋 Project Summary

The repository contains multiple analytical reports covering different domains:

1. **Income and Expenditure Analysis:** A financial tracking dashboard designed to monitor cash flow, categorize spending, and visualize savings trends over time.
2. **Seller Data Analytics:** A retail-focused dashboard that analyzes sales performance, regional distribution, and top-performing product categories.
3. **Sales Analytics (Pivot & Dashboard):** A deep dive into sales data using Power Query for ETL (Extract, Transform, Load) and Pivot Tables for dynamic summarization.

## 🛠️ Technical Features

Each dashboard is built using a combination of the following technical skills:

* **Data Cleaning:** Using **Power Query** to handle null values, format dates, and merge datasets.
* **Dynamic Interactivity:** Implementation of **Slicers** and **Timeline** filters for user-driven exploration.
* **Advanced Calculation:** Utilization of **DAX (Power BI)** and **Excel Formulas** (VLOOKUP, INDEX-MATCH, SUMIFS) for KPI tracking.
* **Visual Storytelling:** Customizing charts (Heatmaps, Treemaps, and Gauges) to highlight key business drivers.

## 📖 How to Use

1. **Download:** Clone this repository or download the specific `.pdf` or `.csv` files.
2. **Excel:** Open the data files in Microsoft Excel to view the pivot structures.
